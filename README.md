# Link Extraction - Расширение для Chrome

Расширение для Chrome, которое собирает список ссылок из активного окна браузера с поддержкой групп вкладок.

## Возможности

1. **Сбор ссылок из активного окна** - собирает все открытые вкладки в текущем окне браузера
2. **Два формата вывода:**
   - **Список** - каждая страница занимает 3 строки (Title, URL, пустая строка)
   - **CSV** - формат CSV с колонками (Title, URL)
3. **Поддержка групп вкладок:**
   - В формате списка добавляется название группы перед каждой страницей
   - В формате CSV название группы добавляется как первая колонка
4. **Копирование в буфер обмена** - кнопка для быстрого копирования результатов

## Установка

1. Откройте Chrome и перейдите на страницу `chrome://extensions/`
2. Включите "Режим разработчика" в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку с этим расширением

## Использование

1. Нажмите на иконку расширения в панели инструментов Chrome
2. Выберите формат отображения:
   - **Список** - для текстового формата
   - **CSV** - для формата CSV
3. Нажмите "Копировать" чтобы скопировать результат в буфер обмена

## Форматы вывода

### Формат "Список"

Для страниц без группы:
```
Title страницы
URL страницы

```

Для страниц в группе:
```
Название группы
Title страницы
URL страницы

```

### Формат "CSV"

Для страниц без группы:
```csv
Title страницы,URL страницы
```

Для страниц в группе:
```csv
Название группы,Title страницы,URL страницы
```

## Структура проекта

- `manifest.json` - манифест расширения
- `popup.html` - HTML интерфейс расширения
- `popup.js` - логика сбора и форматирования данных
- `popup.css` - стили интерфейса
- `icon16.png`, `icon48.png`, `icon128.png` - иконки расширения
